Get Certified

Submit any GitHub repository. If it meets our criteria, we'll certify it and publish it as a local-first utility on our website — accessible to everyone, running entirely in the browser.

What It Means

A Hallucinated Lab Certified utility is a tool that has been reviewed, verified, and published on our platform. Once certified, it becomes available in the Utilities section — each tool loads on demand into the user's browser, does its job, and unloads cleanly when done.

Certification Criteria

  • ✓ Runs entirely on the user's machine — no cloud dependency
  • ✓ Open source with a permissive license (MIT, Apache 2.0, etc.)
  • ✓ No telemetry, tracking, or data collection
  • ✓ Works offline once loaded
  • ✓ Self-contained — loads and unloads from memory cleanly
  • ✓ Clean, documented codebase with a README

The Process

01

Submit Your Repo

Share your GitHub repository link with us. Make sure it has a clear README explaining what your tool does and how to use it.

02

We Review

We check the code against our criteria — local execution, privacy, open source license, code quality, and that it loads/unloads cleanly.

03

Published

Once approved, your utility gets the Certified badge and goes live on our Utilities page — available to everyone, loading on demand.
