Library

A registry of drop-in capabilities for your own projects. Need face recognition in your app? Local OCR? Peer-to-peer networking? Install the library, pick the feature from your terminal, and wire it straight in — no cloud APIs, no keys, no boilerplate.

From Your Terminal

How It Works

terminal — your-project
$ pip install thehallucinatedlab Successfully installed thehallucinatedlab $ thl list nexuslink p2p networking · available face-recognition planned local-ocr planned $ thl add face-recognition ✓ face-recognition wired into ./your-project
1
Install the library

One package, one command: pip install thehallucinatedlab. That's the whole toolchain — no accounts, no API keys.

2
Pick a capability

Browse the registry from your terminal with thl list and pull exactly the feature your project needs — nothing more.

3
Wire it in

thl add <feature> drops the code and models into your project, ready to import. Everything runs locally.
