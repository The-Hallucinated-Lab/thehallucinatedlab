This interface connects to Ollama running on your machine. No API keys, no cloud, no data leaves your device. Completely local.

2

3
If accessing from thehallucinatedlab.space, set the CORS origin:
set OLLAMA_ORIGINS=https://thehallucinatedlab.space
Then restart Ollama.
